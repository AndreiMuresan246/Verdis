# Day 3 Retrospective

## What went well

* Successfully separated the application into multiple files.
* Built the first custom Flutter screen without copying code.
* Better understanding of Flutter architecture and widget responsibilities.
* Improved confidence in reading Flutter documentation.

---

## Challenges

* Learning spacing and responsive layout techniques.
* Understanding when widgets should be const.
* Beginning to think about reusable UI components.

---

## Lessons Learned

* Flutter development is about composing small widgets together.
* A screen is simply another widget.
* Good architecture starts with separating responsibilities.
* UI design should consider different device sizes from the beginning.

---

## Goals for Day 4

* Learn StatefulWidget.
* Create navigation between screens.
* Introduce the Home Screen.
* Continue improving Flutter fundamentals.
