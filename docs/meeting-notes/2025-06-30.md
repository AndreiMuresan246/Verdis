## Decisions

* Timer logic belongs in SplashScreen state, not in build().
* The splash screen decides when it is finished and when to navigate.
* MaterialApp remains inside app.dart.
* SplashScreen and HomeScreen remain separate screens.
* pushReplacement is preferred for splash navigation because the user should not return to the splash screen using the Back button.
* Documentation will continue to be updated after each development session.
