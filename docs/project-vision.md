# Verdis Vision

## Goal

Verdis is an offline-first mobile application that helps users organize, monitor and care for their indoor plants.

The application combines a personal plant collection, an educational catalogue and IoT sensors into one unified experience.

## Target Audience

- Indoor plant enthusiasts
- Beginners learning plant care
- Smart-home hobbyists

## Core Principles

- Offline First
- No mandatory account
- Clean and intuitive UI
- Expandable architecture
- IoT-ready
